# Mini-HBUT 前端页面结构清单（Stitch 设计参考）

> 用于在 Stitch 上构建统一风格的页面设计。列出每个页面的核心元素、区块和交互。

---

## 全局结构（App Shell）

| 元素 | 说明 |
|------|------|
| 底部 Tab 栏 | 4 个 Tab：首页 / 课表 / 通知 / 我的 |
| 视图切换 | 基于状态的视图切换（非 URL 路由），支持 30+ 异步加载页面 |
| 启动画面 | SplashScreen：品牌 Logo + 加载动画 + 连接状态文字 |
| 更新对话框 | 版本检测 + 多线路下载 + 进度条 |
| Toast 消息 | 全局浮动消息提示（info/success/warning/error） |
| 公告弹窗 | 置顶公告 Markdown 渲染弹窗 |

## 通用 UI 组件（模板组件）

| 组件 | 用途 |
|------|------|
| TPageHeader | 统一页面头部：返回按钮 + 图标 + 标题 |
| TEmptyState | 空状态/加载中/错误状态 占位组件 |
| TCard | 通用卡片容器 |
| TSection | 分区标题 + 内容 |
| TStatusBadge | 状态标签(成功/警告/错误/信息) |
| IOSSelect | iOS 风格下拉选择器 |
| ThemeModuleIcon | 主题化模块图标(SVG) |
| SortableSurface | 可拖拽排序容器 |
| LayoutCollisionFxLayer | 拖拽碰撞粒子特效层 |

## 设计令牌（Design Tokens）

| 类别 | 说明 |
|------|------|
| 颜色 | 主题色(primary/secondary) + 语义色(success/warning/danger) + 表面色(surface/border) |
| 字体 | 系统字体栈 + 可选得意黑/黑体/宋体/楷体/仿宋 |
| 间距 | xs/sm/md/lg/xl/2xl 6级间距 |
| 圆角 | sm/md/lg/xl/pill/full 6级圆角 |
| 阴影 | sm/md/lg 3级阴影 |
| 暗色模式 | CSS 变量主题系统，class="dark" 切换 |

## 交互模式

| 模式 | 说明 |
|------|------|
| 下拉刷新 | 部分页面支持下拉刷新数据 |
| 离线优先 | 优先显示缓存数据，后台静默更新 |
| 自动重试 | 网络错误时自动重试(最多3次) |
| 长按编辑 | 首页/通知页长按进入布局编辑模式 |
| 级联选择 | 电费查询的4级级联下拉 |
| 分页加载 | 空教室/图书查询的分页 |
| 流式输出 | AI 助手的 SSE 流式响应 |
| 扫码登录 | 二维码生成 + 实时轮询状态 |
