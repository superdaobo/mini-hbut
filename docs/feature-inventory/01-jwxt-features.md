## 1. 入口

1. 融合门户热门服务 / 办事大厅「**教管一体化平台**」
2. CAS：`authserver/login?service=https://fysso.chaoxing.com/cassso/hbutsie`
3. 进入超星个人空间 + 学习空间首页（含教务应用中心）

## 4. 与 Mini-HBUT 教务模块对照（已有）

已在 App 且与教务强相关：

- 成绩查询、考试安排、绩点排名、学业情况、培养方案
- 选课中心、空教室、全校课表、校历、课表
- 学校消息（含部分教务通知）

**主要缺口（教务侧）：**

1. 等级考试成绩  
2. 学生评教  
3. 微专业 / 辅修 / 转专业等申请  
4. 学业预警独立入口  
5. 实践教学（实习）第三方入口聚合  
6. 在线考试 / 自测（与「考试安排」不同）
