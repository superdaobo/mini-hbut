## 4. 课程内能力（未逐一点开；按超星标准 + 通知线索推断）

通知与文案显示存在：

- 在线章节测验 / 考试
- 作业下发与截止提醒
- 签到 / 预约签到
- 排考通知

**建议产品覆盖优先级：**

1. 作业列表 + 截止提醒  
2. 考试/测验列表  
3. 课程任务聚合（作业+考试+签到）  
4. 课程章节/进度（与现有 shuake/学习记录协同）  
5. 讨论/小组（低优先级）

App 已有：

- 邀请码入班 + 班级资料（`chaoxing_class`）
- 学习通签到（普通/位置/拍照/二维码/手势）
- 学习记录 / 在线学习相关（更多模块）

## 5. 与教务的边界

同一超星会话内：

- **教务应用中心** → 见 `01-jwxt-features.md`
- **左侧菜单 + 课程生态** → 本文件

产品上可做成：

- 「学习通」Tab：课程 / 作业 / 考试 / 资料 / 签到  
- 「教务」Tab：成绩 / 课表 / 排名 / 评教 / 申请  

而不是强制用户理解「教管一体化」品牌名。
