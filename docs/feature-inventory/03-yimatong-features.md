# 一码通 / 一校通功能清单

> 更新：2026-07-22（**手机端门户补抓完成**）

---

## 0. 关键结论

**完整一码通应用矩阵必须走融合门户的「手机端」。**

| 访问方式 | 结果 |
|----------|------|
| Windows / PC 门户 +「校园一码通(PC端)」 | 「一校通服务平台」内网 `172.16.44.6`；`appList` 常为空 |
| **手机 UA / 移动门户** →「一码通(移动端)」 | 跳转 **`https://code.hbut.edu.cn/...#/pages_home/home`**，应用完整 |

客户端对接与产品规划以 **移动端 `code.hbut.edu.cn`** 为准。

### 抓取路径（已验证）

1. 模拟 iPhone UA + 移动视口登录 `e.hbut.edu.cn`
2. 办事大厅搜索「一码通」→ 点 **一码通(移动端)**（或「校园一码通」）
3. 落地：`code.hbut.edu.cn/?tid=...&orgId=2#/pages_home/home`
4. 全部应用：`#/pages_home/appManage`
5. API：`POST https://code.hbut.edu.cn/server/home/appList`（需登录态 Authorization）

## 1. 首页结构（移动端）

| 区域 | 内容 |
|------|------|
| 顶栏 | 学校名、个人中心 |
| 身份卡 | 姓名（脱敏）、身份、学工号、部门、**账户余额** |
| 快捷应用 | 见下表 type=1/2 的首页应用 |
| 其它 | 消息通知、全部应用入口、人脸采集引导弹窗 |

**首页可见快捷（UI 抓取）：**

- 扫一扫、校园码、交易记录、密码修改、解绑  
- 缴电费、重置密码、人脸采集、余额明细  
- 校园生活服务平台、湖工小实、全部应用、消息通知

## 3. 相关 API（协议线索，无密钥）

| 接口 | 用途 |
|------|------|
| `POST /server/auth/getToken` | 换票 |
| `POST /server/user/info` | 用户/卡信息 |
| `POST /server/home/config` | 首页配置 |
| `POST /server/home/appList` | **应用列表** |
| `POST /server/home/noticeList` | 消息通知 |
| `POST /server/home/cardStyleConfig` | 卡片样式 |
| `POST /server/user/idPhoto` | 证件照 |
| `POST /server/virtualCard/config` | 虚拟卡/校园码配置 |
| `POST /server/home/birthdayMessage` | 生日祝福类 |

现有 App 已使用 `code.hbut.edu.cn` + `one_code_token` 等（校园码/电费/流水），可继续扩展。

## 4. PC 端一校通（仅供对照，非全集）

- URL：`http://172.16.44.6:31165/...`（校内）
- 顶栏：首页 / 应用服务 / 消息通知 / 个人中心
- 首页：余额 + 交易列表（含食堂、购电）
- 应用服务：`appList` 空 → **不要当官方功能全集**
