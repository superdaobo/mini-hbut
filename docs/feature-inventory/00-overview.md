# 三平台功能抓取总览

> 抓取时间：2026-07-22  
> 方式：Playwright MCP，用户已登录新融合门户后只读浏览  
> 范围：教务系统（教管一体化 / 超星教务）、学习通、一码通（一校通 / code）  
> **隐私**：文档中脱敏学号/姓名/身份证等，不记录密码

## 1. 平台真实对应关系（重要）

在湖工大 **新融合门户（`e.hbut.edu.cn`）** 语境下：

| 用户口中的名称 | 门户上的实际入口 | 落地系统 |
|----------------|------------------|----------|
| **教务系统** | 热门服务 / 办事大厅「**教管一体化平台**」 | 超星教务门户 `mitudz.jw.chaoxing.com` / `hbut.jw.chaoxing.com`，嵌在 `i.chaoxing.com` 学习空间内 |
| **学习通** | 同上（教管一体化 SSO 到超星）+ 空间左侧菜单 | `i.chaoxing.com` 个人空间 + 课程/作业/云盘等 |
| **一码通** | **手机端**办事大厅「**一码通(移动端)**」 | **`https://code.hbut.edu.cn` 移动 H5（功能全集）**；PC「校园一码通(PC端)」应用列表常空，勿当全集 |

**说明：**

- 经典域名 `jwxt.hbut.edu.cn`（综合教务管理系统）在本次会话中落到 `/admin/login`，**未随门户自动 SSO 进入**；本科生日常教务能力主要在 **教管一体化（超星）** 中。
- 「教务」与「学习通」在超星体系下是 **同一套 SSO 会话**，左侧菜单偏学习通，应用中心偏教务。
- **一码通必须用手机端门户**；Windows PC 门户进一码通会跳到内网一校通且应用列表为空。

## 6. Mini-HBUT 现状摘要

首页模块分组（`Dashboard.vue`）：

- **教务服务**：成绩、空教室、考试、排名、校历、学校消息、学业、全校课表、选课、培养方案
- **一码通**：校园码、电费、交易记录
- **资源**：图书、地图、资料分享、学习通（资料）、小塔、校园助手

更多模块：学习通签到、学习记录（shuake）等。

详见 [04-gap-matrix.md](./04-gap-matrix.md)。
