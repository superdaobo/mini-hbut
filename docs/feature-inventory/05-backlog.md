## P0（建议优先，高频 + 可复用 SSO）

### 1. 学习通 · 作业中心

- **做什么：** 聚合「我的作业」：课程名、作业名、截止时间、状态；支持按截止排序与红点。  
- **为何：** 官方首页常用区 + 通知流最高频；App 几乎空白。  
- **依赖：** 现有 chaoxing SSO / cookie 持久化。  
- **验收：** 登录后进入可见本学期作业；过期会话可静默恢复或明确提示。

### 2. 学习通 · 考试/测验列表

- **做什么：** 「我的考试」/ 章节测验列表（时间、课程、状态）。  
- **为何：** 与「考试安排」（教务排考）互补；通知里大量考试提醒。  
- **注意：** 与 `exams` 模块命名区分（排考 vs 在线考）。

### 3. 学习通 · 待办 / 任务聚合页

- **做什么：** 作业 + 测验 + 未签到 等一页汇总。  
- **为何：** 官方有「待办」；首页价值极高。

### 4. 等级考试成绩

- **做什么：** 教务应用中心「等级考试成绩」只读查询。  
- **为何：** 明确入口且 App 无；实现面通常小于评教。
