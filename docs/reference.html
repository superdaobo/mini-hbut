路径存在性检查

`website/scripts/test-docs-developer-content.mjs` 会读取本页，并用 existsSync 检查本页关键路径确实存在于仓库。 如果后续文件迁移、重命名或删除，契约会失败，防止引用索引变成陈旧清单。

阅读顺序

  1. 从产品视角查功能：先读快速开始、用户手册，再按教务服务、校园生活、社区与通知、扩展模块、设置与数据深入。
  2. 从架构视角查实现：先读开发者总览，再读架构与数据流、平台与 Tauri、模块系统。
  3. 从发布与风险视角查维护：先读构建发布，再读安全与隐私、故障排查。
  4. 需要定位源码时，先在本页找文件，再跳到对应专题页确认上下文和风险边界。
