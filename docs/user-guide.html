用户文档

用户手册

这是普通用户阅读路径和功能地图。它不展开内部接口或源码实现，而是说明你能从哪里进入、哪些功能需要登录、哪些功能无需登录、遇到加载中、空状态、登录过期、离线时应该如何理解。

普通用户阅读路径

1快速开始先完成安装、登录、首页仪表盘、搜索和底部主 Tab 的理解。2教务服务再按学习场景阅读成绩、个人课表、考试、排名、培养方案、选课和空教室。3校园生活然后阅读电费查询、校园码、图书查询、校园地图、资料分享和交易记录。4社区与通知最后补齐论坛、通知中心、课程提醒、电费提醒和后台任务的使用边界。5设置与数据如果换设备、想导出、想改主题/字体/OCR/云同步，再阅读设置与数据。

首页与全局导航

首页与全局导航由主应用统一管理。底部主 Tab 包含首页、个人课表、论坛、通知中心、我的；从首页或我的页进入成绩查询、电费查询、校园码、图书查询、资料分享、更多模块等二级页面后，返回按钮会回到对应父页面。 首页搜索会聚合服务、课程和资讯，适合快速进入模块或查找当天课程。

功能分类地图

教务服务

需要登录

面向学习和教务查询，包含成绩查询、个人课表、全校课表、选课中心、空教室、考试安排、绩点排名、校历、学业情况和培养方案。

一码通

需要登录

面向校园身份和一卡通数据，包含校园码、电费查询和交易记录。校园码、电费与交易流水都依赖登录后的身份或会话。

校园生活

部分无需登录

面向资源和生活工具，包含图书查询、校园地图、资料分享、天气和资源浏览。图书查询、校园地图、资料分享属于无需登录即可从首页进入的资源类入口。

社区与通知

部分需要登录

面向消息和交流，包含论坛、通知中心、课程提醒、电费提醒和后台任务状态。论坛互动需要登录，通知中心还依赖系统通知权限。

扩展模块

按模块决定

面向更多玩法和模块市场，包含更多模块、模块宿主、超星签到、在线学习入口状态和小游戏模块。

我的与设置

部分需要登录

面向个人资料、设置中心、导出中心、反馈、隐私政策、开源协议、赞助和更新等辅助能力。

通用状态说明

加载中

页面正在从教务、一卡通、远程配置、本地缓存或模块目录读取数据。等待结束后通常会进入结果、空状态或错误提示。

空状态

请求成功但没有数据。例如今日无课程安排、没有考试安排、搜索无结果或当前月份没有交易流水。

登录过期

学校会话失效或 Cookie 不再可用。应用会尝试自动恢复会话或重新登录，失败后需要回到我的页处理账号状态。

离线

网络不可用、教务维护或远程服务异常时，支持缓存的模块会展示离线缓存和同步时间；不支持缓存的功能会提示重试或稍后再查。

常用功能速查

功能入口说明
成绩查询首页 -> 教务服务 / 快捷入口查看成绩、绩点、学分，后续专题会补充成绩分布 Beta。
个人课表底部主 Tab -> 课表查看课程详情、自定义课程、导入导出和云同步。
电费查询首页 -> 一码通 / 快捷入口选择宿舍后查看余额，支持失败重试和缓存兜底。
校园码首页 -> 一码通展示在线或离线二维码，依赖身份与配置状态。
图书查询首页 -> 资源无需登录即可搜索馆藏、筛选并查看详情。
资料分享首页 -> 资源浏览 WebDAV 资料目录，支持预览和下载兜底。
论坛底部主 Tab -> 论坛浏览帖子、发帖、回复、收藏和通知，互动需要登录。
通知中心底部主 Tab -> 通知管理通知权限、课程提醒、电费提醒和测试通知。
更多模块我的 -> 更多进入模块市场、小游戏、超星签到或模块宿主。
