Mini-HBUT 是什么

Mini-HBUT 是一款专为湖北工业大学学生设计的跨平台一站式校园助手， 支持 Windows、macOS、Linux、Android 和 iOS 全平台。

核心功能包括：

  • 成绩查询与 GPA 分析、绩点排名对比
  • 智能课表管理、考试安排自动提醒
  • 电费余额查询与低电量告警
  • 空教室查询、校园地图、图书馆搜索
  • AI 学习助手、校园码生成、云同步
  • 后台静默检查、成绩/考试/电费推送通知

亮点特性

跨平台：一套代码覆盖 Windows、macOS、Linux、Android、iOS 五大平台。
离线可用：三层缓存策略保证断网也能查看已加载的数据。
后台通知：成绩更新、考试提醒、电费告警自动推送到系统通知栏。
云同步：一键同步设置和数据到云端，换设备也能秒恢复。
隐私优先：所有数据本地存储，不收集任何个人信息。
完全开源：GPL v3 协议，代码公开透明。

最新能力（v1.3.6）

后台静默检查
课表静默刷新 + 新成绩检测 + 考试提醒 + 电费监控，启动即执行一次全量检查。
AI 学习助手
接入多家大模型（DeepSeek / Qwen / Gemini / GLM），支持流式回复与 LaTeX 公式渲染。
移动端后台任务
Capacitor Background Fetch 支持进程被杀后周期联网检查，含开机自动拉起。
云同步
基于 Cloudflare Workers + KV 的端到端加密云数据同步，支持设置与学业数据。
资源共享
基于 WebDAV 的校园学习资源共享网络，支持文件上传/下载与在线预览。
按需加载 CDN
pdf.js / xgplayer / KaTeX 等大型库首次使用时自动下载并持久化缓存到 IndexedDB。
