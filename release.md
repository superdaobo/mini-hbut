# Mini-HBUT v1.3.6 更新说明

发布日期：2026-05-09

---

## 🎯 新功能

### 📱 Android 桌面小组件

- **今日课程小组件**：在桌面直接查看当天课程安排，显示节次、课程名、时间、教室
- **电费余量小组件**：实时显示宿舍电费余量，低电量自动提醒
- **即将考试小组件**：显示最近一场考试的科目、时间、地点、座位号及倒计时
- 小组件数据通过通知中心自动同步，无需手动刷新
- 支持跟随 App 主题色动态变化
- 连续节次课程自动合并显示（如 3-4 节合并为一行）
- 自定义课程也会在小组件中显示

### 💰 赞助入口

- "我的"页面新增"赞助"按钮（位于开源说明和更多之间）
- 点击显示微信赞赏码，首次下载后自动缓存

### 🔄 更新弹窗改进

- 点击"立即更新"后显示下载源选择表
- 每个下载源右侧实时测速（绿色/黄色/红色标识延迟）
- 支持本站下载、代理下载 1-4、GitHub 源站共 6 个下载源

---

## 🛠️ 优化与修复

### 考试安排页面

- 时间显示修复：去除重复日期，只保留时间段（如 10:00~12:00）
- 信息改为每行一个彩色胶囊显示（日期/时间/地点/座位号）
- 待考/已考对比度增强：待考卡片左侧主题色边框 + 倒计时标签，已考卡片半透明 + 删除线
- 新增考试统计摘要（待考 N 科 / 已考 N 科）

### 通知页面

- 上课提醒、电费监控、成绩动态的关键信息改为彩色胶囊显示
- 考试列表信息改为独立胶囊（日期/时间/地点/座位号）
- 考试时间格式修复，去除重复日期

### 成绩查询

- 修复 `resolveTeacherName is not defined` 导致页面空白的 bug
- 详情页"录入教师"改为只显示后端原始数据，不再 fallback 到课程教师

### 课表页面

- 课程方块间距缩小（margin 2px→1px，padding 7px→5px）
- 视觉更紧凑，信息密度更高

### 空教室

- 修复移动端一行只显示 1 个方块的问题，改为固定 2 列
- 新增分页加载（初始 50 条 + "加载更多"按钮），避免大量数据导致卡顿

### 电费查询

- 修复 `fetchWithCache is not defined` 导致查询失败的 bug
- 宿舍数据加载新增多重兜底（Gitcode → CDN → ghfast 代理 → GitHub 源站）
- 首次下载后永久缓存，后续不再发网络请求

### 模块加载（合成湖工大等）

- 修复安卓第二次进入报 `ERR_BLOCKED_BY_RESPONSE` 的问题
- 修复 iOS 白屏问题：iOS Tauri 使用本地桥加载模块
- 页面加载时排除代理 URL，避免 X-Frame-Options 阻止 iframe

### 网站首页下载

- 下载弹窗新增完整下载源列表（本站/代理 1-4/GitHub 源站）
- 弹窗增加 `max-h-[80vh]` 限制，手机端可滚动不遮挡关闭按钮

### 版本历史页面

- 每个版本显示具体的北京时间（精确到分钟）

- Tauri Rust 后端新增 `write_widget_snapshot` / `write_electricity_snapshot` / `write_exam_snapshot` / `write_widget_theme_color` 命令
- 修复 `widget_bridge.ts` 与 `schedule_prefetch.js` 的循环依赖
- 修复 Tauri Android 运行时检测（`tauri.localhost` 不再被误判为 Capacitor）
