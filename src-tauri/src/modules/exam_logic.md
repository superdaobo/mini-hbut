# 考试安排查询模块逻辑 (exam.rs)

## 1. 模块概述
`exam.rs` 负责查询学生的考试安排（包括期末考试、补考等）。
对应 Python 后端逻辑。

## 2. 核心结构
*   **Exam**: 考试实体，包含课程名、时间、地点、座位号等。

## 3. 逻辑流程
1.  API: `/admin/kwgl/ksap/queryKsap` 或类似接口。
2.  参数: `xnxq` (学年学期), 和 JQGrid 分页参数。
3.  解析 JSON 返回的考试列表。

## 4. 数据清洗
*   服务器返回的时间字段通常需要分割为 日期 和 具体时间段。
*   考场信息通常包含 教学楼 和 教室号，需拼接为完整地址。
